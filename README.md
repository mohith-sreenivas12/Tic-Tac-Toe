
# Tic-Tac-Toe (Java)

A console-based Tic-Tac-Toe game built using Core Java.

## Features
- Two-player game (Human vs Human)
- Row, column and diagonal win detection
- Draw detection (Game tie)
- Input validation for invalid and occupied cells

## Technologies Used
- Java
- OOP concepts
- 2D Arrays
